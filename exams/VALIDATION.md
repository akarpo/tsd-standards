# Math Exam Validation Report

This document captures three independent validation passes on every Form A
math testing-out exam (`math-k.html` through `math-algebra-2-honors.html`).
The same problems appear on the Form B variants in restructured form;
where Form B introduces new problem instances those instances are
validated separately.

The validation rubric:

- **Pass 1 — Math accuracy.** Every numerical answer is computed by hand
  (or symbolic reduction) and confirmed to be reachable with the data in
  the problem statement. Setups must be unambiguous.
- **Pass 2 — Language / grade appropriateness.** Vocabulary, sentence
  length, and concrete framing are appropriate for the target grade,
  benchmarked against canonical advanced textbooks for that level.
- **Pass 3 — MDE / CCSS + advanced-textbook alignment.** Each problem
  ties to a CCSS-M standard cluster cited in the exam subtitle, and
  rigor matches a final-exam item in an advanced textbook for that
  grade (Singapore Math, Beast Academy, AoPS, Forester, Brown).

Reference texts used as alignment benchmarks:

| Grade | Advanced textbook benchmark |
|-------|------------------------------|
| K | Singapore Earlybird Mathematics K |
| 1–2 | Singapore Primary Mathematics 1A–2B; Beast Academy 2A–2D |
| 3 | Beast Academy 3A–3D |
| 4 | Beast Academy 4A–4D |
| 5 | Beast Academy 5A–5D; Singapore PM 5A–5B |
| 6 | AoPS Prealgebra |
| 7 | AoPS Introduction to Algebra (Pt 1); AoPS Number Theory |
| 8 | AoPS Introduction to Algebra (Pt 2); AoPS Introduction to Geometry |
| Alg 1 | Forester *Algebra I*; AoPS *Introduction to Algebra* |
| Geometry | Jacobs *Geometry*; AoPS *Introduction to Geometry* |
| Algebra 2 | Forester *Algebra and Trigonometry*; Larson *Algebra 2* |
| Algebra 2 Honors | Brown *Advanced Mathematics*; AoPS *Intermediate Algebra* |
